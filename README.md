### Инструкция для развертывания проекта

#### Клонирование проекта:
```
git clone https://github.com/AidarPutilov/hm_19_2.git
cd hm_19_2
```
#### Запуск сервера:
##### Windows:
```
python manage.py runserver
```
##### Linux, MacOS:
```
python3 manage.py runserver
```
#### Запуск страницы:
```
http://127.0.0.1:8000/
```
